# switch
+ switch里的表达式必须可以转为int。case标签也必须是整型常量表达式。
+ 一个case标签不是一个作用域，整个switch才是一个作用域。
    ```cpp
    int i;
    switch (i)
    {
    case 1:
        int i;
        break;
    case 2:
        // 错误， 重复定义
        int i;
        break;
    }
    ```
+ 在case标签里不允许初始化定义的变量。
    ```cpp
    int i;
    switch (i)
    {
    case 1:
        // 错误，显式初始化
        int i = 0;
        // 错误，隐式初始化
        string s;
        break;
    case 2:
        // 错误， 重复定义
        int i;
        break;
    }
    ```
+ 把case标签定义在一个块中，case标签就是一个作用域了，也可以初始化变量了。
    ```cpp
    int i;
    switch (i)
    {
    case 1:
        {
            int i = 0;
            string s;
        }
    break;
    case 2:
        int i;
        break;
    }
    ```
